# 经济数据集成指南

## 📊 数据源说明

### 当前状态
- **默认模式**: 使用基于真实历史趋势的模拟数据
- **真实数据**: 配置 API Key 后可获取实时官方数据

### 数据来源

#### 1. FRED (Federal Reserve Economic Data)
- **提供商**: 美国圣路易斯联邦储备银行
- **数据类型**: 
  - 美联储基准利率 (FEDFUNDS)
  - 消费者物价指数/通胀率 (CPIAUCSL)
  - 失业率 (UNRATE)
  - GDP增长率等宏观经济指标
- **费用**: 完全免费
- **限制**: 每天120次请求
- **更新频率**: 每日更新
- **官网**: https://fred.stlouisfed.org

#### 2. CoinGecko
- **提供商**: CoinGecko
- **数据类型**: 加密货币市场数据（市值、价格等）
- **费用**: 免费（无需API Key）
- **限制**: 每分钟10-50次请求
- **官网**: https://www.coingecko.com

## 🔑 如何获取 FRED API Key

### 步骤 1: 注册账号
1. 访问 https://fred.stlouisfed.org
2. 点击右上角 "Sign In"
3. 选择 "Create Account" 注册（完全免费）
4. 填写邮箱和密码完成注册

### 步骤 2: 申请 API Key
1. 登录后访问 https://fred.stlouisfed.org/docs/api/api_key.html
2. 点击 "Request API Key" 按钮
3. 填写简单的申请表单（说明用途即可）
4. 即时获得 API Key（32位字符串）

### 步骤 3: 配置到应用
1. 在应用中打开 "Economy" 页面
2. 点击 "数据源配置" 选项卡
3. 将 API Key 粘贴到输入框
4. 点击 "保存配置"
5. 系统会自动验证并开始使用真实数据

## 📝 数据说明

### 美联储利率
- **指标**: Federal Funds Effective Rate
- **含义**: 银行间隔夜拆借利率
- **影响**: 影响所有利率水平，是货币政策核心工具
- **正常范围**: 0-6%
- **中性利率**: 约3%（长期均衡水平）

### 通胀率 (CPI)
- **指标**: Consumer Price Index (同比)
- **含义**: 消费者物价指数年度变化率
- **目标**: 美联储目标为2%
- **高通胀**: >4% 需要加息抑制
- **低通胀**: <1% 可能需要刺激

### 失业率
- **指标**: Unemployment Rate
- **含义**: 失业人口占劳动力的比例
- **健康水平**: 3.5-4.5%
- **充分就业**: 约4%
- **经济衰退**: >6%

### 加密市场
- **数据**: BTC和ETH市值
- **单位**: 万亿美元
- **周期**: 约4年一个牛熊周期
- **关键事件**: 减半、ETF、监管政策

## ⚠️ 注意事项

1. **API Key 安全**
   - 不要分享你的 API Key
   - 不要提交到公开代码仓库
   - 定期更换 API Key

2. **请求限制**
   - FRED: 每天120次请求
   - CoinGecko: 每分钟10-50次
   - 合理使用，避免浪费配额

3. **数据时效**
   - 经济数据通常有1-2天延迟
   - 预测数据仅供参考
   - 投资决策需综合多方信息

4. **免责声明**
