# 数据准确性说明

## 📊 数据来源和更新时间

### 1. 加密货币市场数据 ✅ 实时

**数据源**: CoinGecko API  
**更新频率**: 实时  
**数据类型**: 真实市场数据  

- **总市值**: 实时获取全球加密货币总市值
- **BTC市值**: 实时获取比特币市值和占比
- **ETH市值**: 实时获取以太坊市值和占比

**备份数据**: 历史数据截至 2024年12月

---

### 2. 美联储利率 ⏰ 延迟发布

**数据源**: FRED (Federal Reserve Economic Data)  
**更新频率**: 每月发布，通常有1-2个月延迟  
**数据类型**: 官方历史数据  

**当前可用数据**: 截至 **2024年12月**
- 最新利率: 4.33%
- 数据日期: 2024年12月

**说明**: 
- 官方经济数据通常有发布延迟
- 2025年及以后的数据需要等待官方发布
- 可通过 GitHub Actions 自动更新（需配置 FRED API）

---

## 🔄 数据更新机制

### 自动更新（推荐）

通过 GitHub Actions 自动从 FRED API 获取最新数据：

1. 配置 FRED API Key（免费）
2. 设置 GitHub Secrets
3. 每天自动运行更新脚本
4. 数据存储在 GitHub Gist

详见: [FRED_API_SETUP.md](./FRED_API_SETUP.md)

### 手动更新

如果不配置自动更新，需要手动更新 `src/utils/economicDataApi.ts` 中的备份数据。

## ⚠️ 重要说明

### 数据准确性原则

1. **只显示真实历史数据** ✅
   - 所有显示的数据都是已发布的官方数据
   - 不包含任何预测或推测数据

2. **数据延迟是正常的** ⏰
   - 官方经济数据通常有1-2个月的发布延迟
   - 这是所有经济数据平台的共同特点

3. **实时数据优先** 🔄

## 💡 建议

如果你需要最新的经济数据：

1. **配置 FRED API** (推荐)
   - 免费申请 API Key
   - 设置 GitHub Actions 自动更新
   - 每天自动获取最新数据

2. **手动更新**
   - 访问 [FRED 官网](https://fred.stlouisfed.org)
   - 下载最新数据
   - 更新 `economicDataApi.ts` 文件

3. **接受延迟**
   - 官方数据本身就有延迟
   - 1-2个月的延迟是正常的
   - 对于长期趋势分析影响不大
